dDNS-ng

dynamic DNS client/server with MySQL support

Download .zip Download .tar.gz View on GitHub

About

If you are tired searching for another free dynamic DNS Service or none of available subdomains doesn't fits you and you are desperate to run your own service, here it is! dDNS-ng solution.

How does it work?

Client connects over the internet to dDNS-ng server which asks database to authenticate client and performs check if client is authorized to update existing record or create a new one. If he will recive positive results from database, then performs necessary actions on DNS server.
